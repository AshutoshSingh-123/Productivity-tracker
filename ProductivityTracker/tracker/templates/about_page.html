About Us

In the Winter Season of Code 2020 hosted by Open Source Club,VIT-AP, The team of WSoC2001 built a production level application(Productivity Monitoring Tool) that tracks user data to provide an in depth analysis of user activity on their computer i.e., desktop apps used, to provide statistical information & also blocks apps/websites to help the user stay more focused during his/her work time.

The project has been implemented solely by using Python, Django, HTML, CSS, Bootstrap and React. This project wouldn't have been completed on time without the guidance of S Vijay Balaji and the contributions made by Taha Junaid, Ashutosh Singh, Vineet Tiwari and Susmita Pal.

Features

We help you stay more focused during work.

Tracks activity

Helps to track every activity of the user.

Analyse Data

Analyse user activity data and present it to the user in more understandable form.

Focus mode

Helps to put a control on apps used by users.

Scheduler

Can be used to schedule focus mode.
